Welcome to **Insights** — a space for posts, ideas, and thoughts on technology, design, and building things.

## What You'll Find Here

Insights covers a range of topics:

- Technical explorations and tutorials
- Thoughts on software development practices
- Notes on tools and technologies I use
- Ideas and perspectives on building products

## Format

Posts are written in MDX, enabling:

- Clear structured content with semantic headings
- Code examples with syntax highlighting
- Callouts for important notes
- Links and references

### Example Code

```typescript
interface Post {
  title: string;
  slug: string;
  tags: string[];
  date: string;
}

const createPost = (data: Omit<Post, 'slug'>): Post => ({
  ...data,
  slug: data.title.toLowerCase().replace(/\s+/g, '-'),
});
```
